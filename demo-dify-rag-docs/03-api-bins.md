# API-001. API ячеек хранения

## POST /bins

Создаёт новую ячейку хранения.

Request body:

```json
{
  "code": "A-001"
}
```

Ошибки:
- 400, если code не указан;
- 400, если code состоит только из пробелов;
- 409, если ячейка с таким code уже существует.

## GET /bins

Возвращает список созданных ячеек хранения.
