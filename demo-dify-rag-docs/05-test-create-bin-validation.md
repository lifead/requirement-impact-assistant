# TEST-001. Проверка валидации создания ячейки

Сценарий: создание ячейки хранения без корректного code.

Проверки:
- если code отсутствует, API возвращает 400;
- если code равен пустой строке, API возвращает 400;
- если code состоит только из пробелов, API возвращает 400;
- если code уже существует, API возвращает 409.

Ожидается, что ошибка является управляемой и не приводит к HTTP 500.
