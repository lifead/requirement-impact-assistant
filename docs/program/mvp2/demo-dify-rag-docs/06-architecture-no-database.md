# ARCH-001. Архитектурное ограничение MVP

В рамках MVP запрещено добавлять базу данных, ORM, миграции и фоновую синхронизацию.

Все изменения должны сохранять простую архитектуру:
- ASP.NET Core Minimal API;
- in-memory storage;
- тесты на уровне API/сервиса;
- без внешней инфраструктуры хранения.
