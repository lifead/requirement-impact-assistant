# REQ-001. Создание ячейки хранения

Система должна позволять создать ячейку хранения.

Ячейка хранения имеет обязательный код. Код должен быть уникальным в пределах системы.

Если код не указан или состоит только из пробелов, система должна вернуть ошибку валидации.

На этапе MVP создание ячейки не требует подключения базы данных.
