# MVP-1 stabilization

Папка фиксирует stabilization items после MVP-1.

Это не этап MVP-2 и не место для расширения функциональности. Здесь ведутся только небольшие стабилизационные работы, найденные после сборки MVP-1: дефекты, локальные риски воспроизводимости, уточнения проверок и технические долги, необходимые для устойчивой демонстрации.

Правило ведения:

- один stabilization item - один отдельный файл;
- общий список ведется в [stabilization-index.md](stabilization-index.md);
- каждый item должен иметь понятный статус, область влияния, границы scope и ожидаемые проверки.
